Instructions:

Keyboard:

Left/Right - Steer

Up - Thrusters

Z or Space - Fire

R - Restart

Controller (PS4):

D-pad - Steer

R1 / X - Thrusters

L1 / Square - Fire

Options - Restart

Controller (Xbox):

D-pad - Steer

RB / A - Thrusters

LB / X - Fire

Start - Restart
